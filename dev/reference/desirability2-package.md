# In-line desirability functions

Desirability functions (Derringer and Suich, 1980,
[doi:10.1080/00224065.1980.11980968](https://doi.org/10.1080/00224065.1980.11980968)
) can be used for multivariate optimization.

## Details

See:

- [`d_max()`](https://desirability2.tidymodels.org/dev/reference/inline_desirability.md)
  for individual desirability functions.

- [`d_overall()`](https://desirability2.tidymodels.org/dev/reference/d_overall.md)
  for combining functions into an overall objective function.
